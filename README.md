# MyArchConf

En este repositorio encontrarás mis archivos de configuración (dotfiles) y configuraciones de seguridad para mi distribución Arch Linux + Hyprland.

> [!IMPORTANT]
> Este repositorio no es una guía de instalación, sino más bien un respaldo y referencia para mi configuración personal.

## Estructura del Repositorio

A continuación se detalla el contenido de las carpetas principales:

- **`config/`**: Contiene los archivos de configuración (dotfiles) para diversas aplicaciones y el entorno de escritorio.
  - `hypr/`: Configuración de Hyprland (gestor de ventanas).
  - `waybar/`: Configuración de la barra de estado.
  - `kitty/`: Configuración del emulador de terminal.
  - `rofi/`: Configuración del lanzador de aplicaciones.
  - Y otras configuraciones para `fastfetch`, `btop`, `cava`, etc.

- **`wallpapers/`**: Colección de fondos de pantalla utilizados en mi configuración.

- **`install-scripts/`**: Scripts de utilidad e instalación para desplegar el entorno.

- **`security/`**: Documentación y notas sobre configuraciones de seguridad (SSH, UFW, etc.).
